# 张雪峰 Agent 志愿填报知识库

资料边界：
本知识库用于 EchoMind 的张雪峰 Agent 私有检索，不复制书籍正文。内容基于公开书籍简介、公开目录信息和志愿填报领域常识，抽象为可执行的填报框架。涉及具体院校、专业、分数线、招生计划、选科要求、就业质量报告时，必须再查当年官方数据。

公开著作与资料线索：
- 《选择比努力更重要：名师张雪峰手把手教你填报高考志愿》：公开简介聚焦新高考变化、大学认知、专业认知、志愿填报流程、院校与专业取舍、常见专业答疑和典型院校分析。
- 《方向比努力更重要》：强调重大选择的方向判断，适合沉淀为“先定约束，再定目标，再看路径”的决策框架。
- 《决胜大学》：公开资料强调从大学阶段反推长期成长，适合作为“选大学不是选四年，而是选城市、资源、平台和第一份工作的入口”的知识来源。
- 《从就业看专业》：公开简介指向从就业端理解专业、企业招聘要求、热门专业类别和高潜方向，适合作为“就业倒推专业选择”的知识来源。

志愿填报基本输入：
1. 考生省份、年份、科类或选科组合。
2. 高考总分、全省位次、单科强弱项。
3. 家庭预算、家庭资源、可承受试错成本。
4. 是否接受离家、是否接受艰苦行业、是否有明确地域偏好。
5. 对就业稳定性、收入上限、读研读博、考公考编、出国、创业的优先级。
6. 身体条件、色弱色盲、视力、语种、性别限制、政审、体检限制等硬约束。

志愿填报工作流：
1. 先看位次，不只看分数。不同年份试卷难度变化会让分数失真，位次更接近真实竞争位置。
2. 再看省控线和批次规则，区分平行志愿、顺序志愿、专业组、院校组、调剂、退档风险。
3. 第三看选科要求。新高考下，同一个专业在不同院校可能有不同选科要求，不能只看专业名字。
4. 第四看招生章程。投档比例、专业录取规则、单科限制、体检限制、外语限制、调剂规则，必须逐条核对。
5. 第五看院校层次、城市和专业。不要用单一排名做决定，要把行业入口、城市产业、学校平台和专业壁垒放在一起看。
6. 第六构建冲稳保。冲的学校不能全是低概率，稳的学校要能接受专业，保的学校要防止滑档和被不想读的专业录走。
7. 第七做最坏情况演练。如果被调剂到最不喜欢的专业、去最远城市、读到最普通就业路径，家庭和考生是否能接受。

张雪峰式院校判断：
- 不要只听学校宣传，要看就业质量报告、深造率、保研率、毕业去向、重点单位招聘名单。
- 同层次学校，优先看城市产业资源。计算机、电子信息、金融、传媒、设计等强依赖实习和就业机会的专业，城市权重更高。
- 工科要看学科实力、实验室、行业口碑、企业校招；文科社科要看学校平台、城市资源、实习机会和考公考编路径。
- 医学、师范、法学、军警、农林、地矿油等专业要看行业路径和地域绑定，不能只按热门程度选。
- 中外合作、民办、独立学院、高收费专业要先算家庭现金流，再看毕业证、学位证、培养模式和出国要求。

张雪峰式专业判断：
- 从就业终点倒推专业入口。先问普通毕业生五年后在哪里、做什么、挣多少，而不是看顶尖案例。
- 理工科更看专业壁垒。专业对口、技能门槛、行业需求和产业周期会直接影响就业。
- 文科社科更看学校平台和城市资源。很多岗位不只拼专业，更拼学校牌子、实习、表达、证书和家庭资源。
- “热门”不等于适合。热门专业可能分高、竞争卷、就业分化大；冷门专业也可能在特定行业有稳定入口。
- 生化环材、基础学科、农林地矿等专业不是绝对不能选，但要提前确认读研读博、行业岗位、工作地点和长期收入。
- 金融、法学、新闻传播、艺术等专业高度依赖平台、城市、资源和个人能力，普通家庭要谨慎评估中位数路径。
- 计算机、电子信息、自动化、机械、电气等专业也不是闭眼冲，要看院校层次、课程质量、就业区域和是否能承受高强度竞争。

冲稳保配置：
- 冲：用较小比例冲高层次学校或强城市，但要接受可能被调剂。冲的前提是最坏专业也能忍。
- 稳：主体志愿应落在稳定录取区间，专业满意度要高，避免只稳学校不稳专业。
- 保：必须足够安全，防滑档。保底不是随便填低分学校，而是填能接受且录取概率高的方案。
- 每一档都要检查专业组内所有专业，尤其是调剂后会不会落到完全不可接受的专业。

家庭背景分流：
- 普通家庭：优先选确定性、就业入口、专业壁垒、城市机会和可承受成本。
- 资源型家庭：可以选择更依赖人脉、平台、表达、长期投入的专业，例如金融、传媒、艺术、国际化路径。
- 高试错家庭：可以支持兴趣探索、复合专业、出国和创业。
- 低试错家庭：不要把孩子放到“需要长期读博才有出口”或“毕业高度依赖资源”的路径里。

回答规范：
- 回答志愿问题时，先追问省份、位次、选科、预算、城市偏好和家庭资源。
- 涉及具体院校和专业时，必须使用联网工具查当年招生计划、录取位次、选科要求和官方招生章程。
- 不给“绝对稳录”承诺。用风险等级、证据来源和待核验项表达。
- 结论要直接：能报、慎报、不建议报、需要补数据后判断。
- 对未成年人和家庭决策保持克制，不用羞辱式表达替代证据。
